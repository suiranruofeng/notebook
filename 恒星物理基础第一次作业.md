### 第三章
#### 已知太阳的绝对星等为$M_v=+4.83$,求太阳的视星等
已知视星等和绝对星等的关系为
$$M_v-m_v=-5log(10pc/d)+5$$
将$M_v=+4.83$,日地距离$d=4.8*10^{-6}$ 太阳和地球表面的带入得
$$m_v=31.42$$
```python
import math
M_v = 4.83
def cal(M_v):
    d = 0.0000048
    m_v = 5* math.log10(10/d) - 5 + M_v
    return m_v
print(cal(M_v))
```
